این پروژه یک سیستم مدیریت آموزشگاه آنلاین (LMS) است که با استفاده از:

✅ فرانت‌اند: React + TypeScript + Vite
✅ بک‌اند: FastAPI (Python)
✅ پایگاه داده: PostgreSQL

پیاده‌سازی شده است. سیستم دارای سه نقش کاربری است:
- مدیر (Admin): مدیریت کامل سیستم
- مدرس (Teacher): ایجاد دوره، ثبت نمره، حضور و غیاب
- دانشجو (Student): ثبت‌نام، پرداخت، مشاهده نمرات و گواهی

کل پروژه به صورت کامل به یکدیگر متصل است و تمام عملیات CRUD از طریق API انجام می‌شود.

================================================================================
                            قابلیت‌های اصلی
================================================================================

1. احراز هویت JWT با سه نقش (Admin، Teacher، Student)
2. مدیریت دوره‌ها (ایجاد، ویرایش، حذف، جستجو)
3. مدیریت کاربران (مشاهده، تغییر وضعیت، حذف منطقی)
4. سیستم ثبت‌نام و پرداخت
5. سیستم تکالیف و نمره‌دهی
6. سیستم حضور و غیاب
7. صدور گواهی پس از اتمام دوره
8. سیستم اطلاعیه‌ها
9. گزارش‌گیری پیشرفته (۹ گزارش مختلف)
10. کارنامه تحصیلی با محاسبه GPA

================================================================================
                            جداول اصلی پایگاه داده
================================================================================

جدول‌های اصلی:

- users          : اطلاعات کاربران (Admin، Teacher، Student)
- courses        : دوره‌های آموزشی
- enrollments    : ثبت‌نام دانشجویان در دوره‌ها
- assignments    : تکالیف هر دوره
- submissions    : پاسخ‌های ارسالی دانشجویان
- payments       : تراکنش‌های مالی
- attendance     : حضور و غیاب
- certificates   : گواهی‌های صادرشده
- announcements  : اطلاعیه‌ها
- logs           : لاگ عملیات سیستم

================================================================================
